## 기본명령
### 화면 나누기
``` 
:sp.	
:split .	
```
화면을 가로로 2등분한다.

```
:vs.	
:vsplit
```
화면을 세로로 2등분한다.

```
:e.	
:edit .	
```
워킹 디렉터리를 기준으로 파일익스플로러를 연다.

```
:E	
:Explore
```
현재 파일을 기준으로 파일익스플로러를 연다.

```
:Se	
:Sexplore
```
화면을 가로로 등분하여 파일익스플로러를 연다.

```
:Vex	
:Vexplore	
```
화면을 세로로 등분하여 파일익스플로러를 연다.
